Extract the given tarball and cd to the 360t directory.
Where is code?
The cpp files are in src and .h files are under in include directories.

How to compile -
from 360t dir, execute - cmake . followed by make

How to run -
from 360t dir, run following scripts -
1. bash run_multiprocess.sh - demo of 2 player processes sending messages.
2. bash run_multithreaded.sh - demo of 2 player threads sending messages.

Note: The scripts are not made executable for security reasons.
